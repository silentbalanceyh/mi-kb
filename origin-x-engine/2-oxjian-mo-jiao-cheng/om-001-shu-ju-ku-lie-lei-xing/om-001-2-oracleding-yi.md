# Oracle定义

## 1. 列类型

| 模型类型 | 列类型 | 含义 |
| :--- | :--- | :--- |
| BOOLEAN | NUMBER\(1\) | 布尔值 |
| INT | NUMBER | 整数 |
| LONG | NUMBER | 长整数 |
| DECIMAL | NUMBER | 精度浮点数（货币） |
| DATE1 | DATE | 日期格式 |
| DATE2 | DATE | 日期/时间综合格式 |
| DATE3 | TIMESTAMP | 时间戳格式（一般整数） |
| DATE4 | x | 时间格式 |
| STRING1 | VARCHAR2 | 带长度字符串 |
| STRING2 | VARCHAR2 | 长文本 |
| JSON | CLOB | JSON格式 |
| XML | CLOB | XML格式 |
| SCRIPT | CLOB | 脚本引擎存储的脚本格式 |
| BINARY | BLOB | 二进制数据 |
